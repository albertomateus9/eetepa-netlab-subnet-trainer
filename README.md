# EETEPA NetLab Subnet Trainer

Visual CIDR, VLSM, VLAN, and IP addressing trainer for technical networking classes.

Live demo target: https://albertomateus9.github.io/eetepa-netlab-subnet-trainer/

## Overview

EETEPA NetLab Subnet Trainer is a public educational portfolio MVP inspired by EETEPA Vilhena Alves. It connects technical computing, networking, and data science learning through a local-first browser app.

This is not an official institutional system. It uses synthetic/demo data only and does not publish student data, internal school documents, or personal contact details.

## What It Demonstrates

- Applied learning for technical education.
- Local-first browser workflows with no backend dependency.
- Exportable outputs for classroom discussion and documentation.
- A practical bridge between computing, networking, and data science.

## Tech Stack

- React + Vite + TypeScript
- Browser APIs and local state
- Synthetic datasets
- GitHub Pages-ready static build

## Getting Started

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
```

## Privacy And Safety

- No backend service is used by the app.
- No real student, teacher, or institutional record is included.
- Uploaded files are processed locally in the browser.
- Hardware features, when present, include synthetic demo mode.

## Roadmap

- Add classroom activity guides.
- Add automated tests for the core calculation helpers.
- Improve mobile interactions and accessibility.
- Add optional integrations only when they preserve local-first behavior.
